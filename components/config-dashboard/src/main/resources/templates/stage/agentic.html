How to use Stage 7 demos

Each demo lets the LLM drive the conversation through a forced send_message tool call. Demo 01 replies once with visible inner thoughts; demo 02 loops up to 5 steps, deciding itself when to stop.

  1. Make sure both cards below show a green status pill (start them via ./workshop.sh agentic start all if needed).
  2. Click Open console on a card — an inline chat panel appears below it.
  3. In the left rail, type an agent id and hit Create.
  4. Select the agent in the list, type a message in the center textarea, hit Send.
  5. Click ▸ inner thoughts on any bubble to reveal the model's private reasoning.

What is an agent id? It's the label for one conversation — like a chat tab name. Each id has its own independent chat memory on the server, so alice and bob never see each other's history. Pick any short string; create as many as you like; use Reset memory to clear one, Delete to forget it entirely.

Click an action button on any agent card to see the result here.
